# Territorial Defense: The Game
:::warning
*Підтримку проєкту зупинено*
:::

![Промо банер Territorial Defense: The Game](../../../../../static/img/terdef-banner.png)

Влітку 2022, після почтаку повномасштабного російського вторгнення в Україну, разом з моїм другом і
колишнім партнером з Stub Games ми почали працювати над мобільною грою у жанрі крафтового tower defense.
Ця гра не стала успішною, але я все ще люблю її. Я написав щось на кшталт пост-мортему про неї у своєму блозі,
який називається [Another Drop in the Ocean](https://notpublicopinion.com/another-drop-in-the-ocean/). Нажаль поки тільки англійською.
